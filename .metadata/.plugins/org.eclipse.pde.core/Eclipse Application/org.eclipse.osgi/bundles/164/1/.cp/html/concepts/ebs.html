Elastic Block Storage

Amazon Elastic Block Storage (EBS) is a new type of storage designed specifically for Amazon EC2 instances. Amazon EBS allows you to create volumes that Amazon EC2 instances can mount as devices.

Amazon EBS volumes behave like raw unformatted external block devices. They have user-supplied device names and provide a block device interface. You can load a file system on top of Amazon EBS volumes, or use them just as you would use a block device.

Since EBS volumes behave just like any other mounted hard drive you can do anything with them you would do with standard hard drives—even make a RAID array.

After writing data to an Amazon EBS volume, you can periodically create a snapshot of the volume to use as a baseline for new volumes or for data backup.

There is also a growing set of public data sets, such as the Human Genome and US Census data, which you can use to make your own EBS Volumes.
