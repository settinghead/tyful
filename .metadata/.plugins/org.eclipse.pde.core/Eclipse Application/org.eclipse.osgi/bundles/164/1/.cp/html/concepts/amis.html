Amazon Machine Images (AMIs)

An Amazon Machine Image (AMI) is a copy of a server instance that you can use to launch identical copies of the instance at any time. For example, an AMI might contain Linux, Apache, and your web site content; it might contain Linux, Hadoop, and a custom application; or it might contain Windows, Internet Information Server (IIS), SQL, and your web store data.

When launching AMIs, you can select an Amazon-provided AMI, a public AMI that another Amazon EC2 user provided, or an AMI that you create.
