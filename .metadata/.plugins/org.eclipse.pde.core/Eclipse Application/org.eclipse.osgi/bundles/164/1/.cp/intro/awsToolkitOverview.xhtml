AWS Toolkit for Eclipse

Configure your AWS Account

The AWS Toolkit for Eclipse makes getting started with Amazon Web Services easy, but before you can begin, you'll need to configure your account by following the three easy steps below.

  • Create an AWS account
  • Sign up for EC2
  • Configure the AWS Toolkit with your access keys

Using the AWS Toolkit for Eclipse

After you've configured your AWS account, you're ready to start exploring the AWS Toolkit for Eclipse.

EC2 Management Views

The AWS Toolkit provides several management views that make working with EC2 easy. You can find these views under the Amazon EC2 category in the Show View dialog.

  • Open the AWS Toolkit views

Web Tools Platform Integration

The AWS Toolkit for Eclipse allows you to use the standard WTP tools to deploy and debug Java web applications to a group of Tomcat servers running in EC2.

It's as easy as creating a new Amazon EC2 Tomcat Cluster through WTP's New Server Wizard and running your Java web application with the Run on Server menu item.

  • Create an Amazon EC2 Tomcat cluster
  • Web Tools Platform homepage
